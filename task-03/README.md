# WireFrame development for **MeloFi**

After my MeloFi SRS document got approved by my mentors, I pursued to develop the low-fidelity WireFrame for MeloFi, for both mobile and web versions.

I used Figma to develop the WireFrame. It's the first time I am using Figma for WireFraming applications.

You can view my low-fidelity MeloFi WireFrame design done through Figma from here:
